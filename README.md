# PlayTogether
一款以兴趣为驱动的陌生人社交app

* 后台服务器使用的 leancloud，通过 rxJava 对其 sdk 进行封装 
* 单独抽取一层 controller 层，避免 activity 过于臃肿
* 对聊天模块以及相册模块单独封包，便于以后提取复用
* dagger2 依赖注入
* EventBus
